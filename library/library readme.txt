These libraries are to be installed before the exceution of program

steps

1) Open your arduino ide

2)Go to "sketch" on the top menu

3)Then sketch <  Import library > add library 

4) Window pops up to select the folders

5) Then select the folder AcceleroMMA7361 from library folder

6) Repeat the same process to select folder Adafruit_NeoPixel

7) Done !!!

8) Execute and enjoy
